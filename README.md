# Registration system

### Установка
Склонировать проект:
```bash
git clone https://github.com/zheki4sh05/UserRegistrationSystem.git
```
Скомпилировать проект:
```bash
mvn clean package
```

Запустить docker-compose:
```bash
docker-compose up
```

### Использование
Приложение доступно по адресу
```bach
http://localhost:8080
```
